## Installation

Add this line to your application's Gemfile:

```ruby
gem 'url-resolver'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install url-resolver

## Usage

To resolve a partial file path use:

```
UrlResolver::resolve(url, path_to_resolve)
```
